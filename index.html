Application Creating Life Cycle


Stages for creating a application project

Developing an application is look like constructing a block-structured lego toy. We can handle the most features of application well with block extensible patterns.

The first important thing is to write block extensible each other .

The second important thing is to have a good clean architecture in which all block features can well connect each other indepentently.

  2. Memorize them

  3. Go to Github , find your appropriate code repositary to clone your skill and proficiency.Download and clone what you like.
