What is List ?

Data ကနေပြောပါမယ်။ Programming ဆိုတာ data တွေနဲ့ computing logic လုပ်ရတာဖြစ်တဲ့တွက် data တေွလိုတာပေါ နော်။ Data တွေကိုအမျိုးအစားခွဲခြားလိုက်ရင် integer ,float , double , string စသဖြင့် data types လေးတွေရလာတယ် ။ ဥပမာ - integer ဆိုရင် ဂဏာန်းလေးတွေပေါ့ ၊ string ဆိုရင် နေ့စဥ်အသုံးပြုနေ တဲ့စာသားတွေ text တွေပေါ့။ အဲဒီ data တွေကို memory မှာ Save လိုက်မယ် ပြီးတော့ပြန်ခေါ် သုံးမယ်ဆိုရင် နာမည် တစ်ခုတော့‌ သေချာပေါက်လိုမှာပေါ့နော် ။ လူတစ်ယောက် အသားဖြူဖြူ အရပ်ရှည်ရှည်နဲ့ ကိုယ်နဲ့ တစ်တန်းတည်းထိုင်တဲ့ ကောင်လေး ကို အိမ်ရောက်တော့ မိသားစုကို သူအကြောင်းပြောချင်ရင် အမေရေ "လူတစ်ယောက် အသားဖြူဖြူ အရပ်ရှည်ရှည်နဲ့ ကိုယ်နဲ့ တစ်တန်းတည်းထိုင်တဲ့ ကောင်လေး" လို့ နာမည်လေး မသတ်မှတ်ပဲသုံးရင် ပြောရတာခရီးမတွင်တော့မှာ သေချာပါတယ်။ ဒါကြောင့်သူ့ကို "မောင်ဖြိုးဖြူ" လို့ နာမည်လေး ပေးလိုက်တယ်။ အဲဒါဆို သူအကြောင်းပြောလို့လွယ်ကူသွားပြီ ။ အဲဒီလိုပဲ save ထားတဲ့ data လေးတွေကို လဲ မ save ခင်ကတည်း နာမည်လေးတစ်ခုသတ်မှတ်လိုက်ရင် ကိုယ်လိုအပ်လို့ပြန်သုံးချင်တဲ့အခါ လွယ်ကူသွားမယ်။ အဲဒီ နာမည် လေးကို variable name လို့ခေါ်တယ် ။ သူက data ကို ညွန်းပြပေးတဲ့ ကိုယ်စားပြု နာမည်လေးတစ်ခုပေါ့လေ။


Variable တွေက data pointer ( Point = ညွန်းဆိုသည်) လို့သိပြီးသွားပြီဆိုရင် ‌int a = 10 ; မှာ a သည် 10 ရှိတဲ့ memory address လေးကိုသူက သိမ်းပေးထားပါတယ်။ ဥပမာ - မောင်ဖြိုးဖြူရဲ့ နေရပ်လိပ်စာ လေးပေါ့ သူကိုတော့ code တေွထဲ မှာမ မြင်ရပါဘူး မြုပ်ထားတာပေါ့နော်။ သို့သော် programmer က '‌a' ကိုသုံးလိုက်တာနဲ့ code တွေက လိပ်စာရှာပြီး a ရဲ့ လိပ်စာအတိုင်း save ထားတဲ့ 10 ဆိုတဲ့ value လေးကို ကိုယ်ဆီပြန်ယူလာပြီး တွက်ချက်ပေးမှာပေါ့နော် ။


ကဲ.. Variable အကြောင်းသိပြီ ဆိုရင် နောက်တဆင့်သွားကြရအောင်။ အကယ်၍ တခါတလေ a , b , c , d , e , f , g ,etc., အစ သဖြင့် integer data type လေး တွေအများကြီးကို တွက်ချက်လို့လိုလာပြီ ။ သူတို့ ကို a = 10 , b = 11 , c = 20 , d = 22 , စ သဖြင့် သတ်မှတ် နေရင် အချိန်တေွကုန်တာပေါ့ ပြီးတော့ code memory တွေလဲပိုတက်လာ မှာပေါ့နော်။ အဲဒီလို multiple values တွေကို တစ်စုတည်းစုပြီး data type ‌လေးတစ်ခုဖန်တီးလိုက်ကြတယ် ။ သူက တော့ data structure type ( data collection) လေးပါ ။ Data Structure ဆိုတာ သူကိုယ်က Type တစ်ခုပဲ သို့သော် သူက value တွေကို အစီအစဥ်နဲ့ အတန်းလိုက်အများကြီး save နိုင်တယ် ပြန်ထုတ်သုံးရင်လဲ အတန်းလိုက် ပြန်ထုတ်သုံးနိုင်တဲ့တွက် လျင်မြန်လွယ်ကူတာပေါ့။ သူကို array data type လို့ခေါ် တယ်။


Programming Languages ( C,C++, Java ,Python,etc) တေွသူတို့ကိုယ်ပိုင် နာမည်တွေမှည့်ပြီး array ကို အမျိုးမျိုးခေါ် ‌ ‌ေ၀ါ် ကြပါတယ်။ Python မှာဆို array ကို List လို့ ခေါ် သလို Java မှာလဲ Array လို့ခေါ် ကြတယ် ။ သို့သော် အလုပ်လုပ်ပုံက တော့ အတူတူပဲ ဖြစ်ပါတယ်။ ဥပမာ - List data = [ 1 , 4 , 5 , 6 , 7 , 8 ] ; သူတို့ကို ပြန်ထုတ်သုံးမယ်ဆိုရင် index လေးတေွနဲ့ ပြန်ထုတ်သုံးကြတယ် data[0] = 1 , data[1] = 4 , data[2] = 5 ။ List တစ်ခုမှာ Indexing ဆိုတာက ပါ၀င်တဲ့ data member လေးတွေကို အမှတ်စဥ်သတ်မှတ်ပေးတာပေါ့ ပြန်ယူသုံးဖို့နဲ့ သိမ်းဆည်းဖို့ပေါ့နော်။ data collection ( data structure ) တေွအားလုံးရဲ့ အမှတ်စဥ်က အမြဲ index = 0 ( Start Point ) ကနေစကြပါတယ်။


Data Structure တွေမှာ တခါတလေ ကိုယ်က number type တစ်ခုပဲသီးသန့် save ချင်ရှိသလို , တခါတလေလဲ number ရော text ( string type ) ရော ပေါင်း save ချင်တာရှိကြပါတယ် ။ အမှန်ဆို data structure တစ်ခုက number data structure ဆိုရင် number ပဲ save လို့ရပါတယ် သို့သော် နောက်ပိုင်းကျတော့ စာသားတွေပါတစ်ခါတည်း ရော save ချင်တဲ့ အခါတွေ ရှိလာ တော့ data type အသစ်ကို ဖန်တီးလိုက်ကြတယ် ၊ သူက dynamic data type လို့ခေါ် တယ်။ dynamic data type ဆို number လဲဖြစ်နိုင်တယ် ၊ စာသား (string) လဲဖြစ်နိုင်တယ် ၊ ဒဿမကိန်းလဲ ဖြစ်နိုင်တယ် အဲတော့ Array ရဲ့ type လေးကို ကိုယ်က dynamic type လို့ပေး ထားရင် ဘာ data type ပဲ ဖြစ်ဖြစ်အဆင်ပြေ သွားတာပေါ့။ Array တစ်ခုရဲ့ type ကိုကြေငြာချင်ရင် List data = [ 'text' , 1 , 19.39 ] ဆိုပြီး မြားလေး နှစ်ခုကြားမှာ Type ကို ကြေငြာပေးရပါတယ် ။
